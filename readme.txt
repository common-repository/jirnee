== Description ==

Help your visitor promote your travel site! The Jirnee- Travel Share bookmarking Widget allows any visitor to bookmark your travel site easily with Facebook as the current service (with the goal to add more services shortly). Sign up for a Jirnee.com account to see how your visitors are sharing your travel content, which destination is shared the most, and more. It's all free--even the pretty charts and graphs.




== Installation ==

This section describes how to install the plugin and get it working.



1. Unzip the folder jirneeshare.zip 
2. Upload `jirneeshare.php`, 'idg_admin.php' to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Under the settings you will find jirnee-settings where you can enter the API key to activate the plugin. 
5. Go to www.jirnee.com and signup to find the api key
